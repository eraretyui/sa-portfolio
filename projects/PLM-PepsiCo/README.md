# PLM-система для ООО «ПепсиКо Холдингс»

## Описание проекта
**Объект исследования:** ООО «ПепсиКо Холдингс»  
**Период:** 2025–2026  
**Контекст:** Курсовая КНИР + практические работы, НИТУ МИСиС

## Проблема
Процесс разработки новой рецептуры продукции велся в разрозненных системах:
Excel-таблицы, локальные БД, ручные согласования. Это приводило к:
- увеличению Time-to-Market
- ошибкам в расчётах и нормативным несоответствиям
- сложностям в управлении версиями данных

## Цель
Спроектировать PLM-систему для автоматизации процесса — от получения
технического задания до формирования итогового пакета документации.
